# Dev Docs

This directory is for site-maintainer notes only.

It is **not published on the website** because Starlight only turns files under `src/content/docs/` into public pages.

Important caveat: if this GitHub repository is public, files in `dev-docs/` are still visible in the repo. They are only excluded from the website build.

## Files

- `site-maintainer-notes.md`: how the site was created, the main technical decisions, and what the content structure is intended to support over time.
