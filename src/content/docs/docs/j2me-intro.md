# J2ME Introduction

Java ME sits in the overlap between constrained hardware and surprisingly ambitious software.

This section is where working notes should go once a setup is repeatable:

- target devices and firmware versions
- SDK versions and download sources
- emulator setup steps
- build, package, and deployment notes

Start small: compile one MIDlet, run it in an emulator, and record the exact toolchain that worked.
