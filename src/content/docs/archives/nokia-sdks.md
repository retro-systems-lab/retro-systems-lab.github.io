# Nokia SDKs

Vendor SDKs disappear quietly, so the archive should focus on preserving the full context around each release.

Archive targets:

- installer files and checksums
- release notes and version numbers
- bundled device profiles
- sample projects
- license text and original documentation

If a toolchain only works on a legacy operating system, document that constraint next to the download record instead of leaving it implicit.
