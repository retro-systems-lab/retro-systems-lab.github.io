J2ME DevKit is a modern development environment for building and running **Java ME (MIDlet)** applications on contemporary machines.

Developing for Java ME today can be difficult because the original toolchains and documentation are fragmented, outdated, or difficult to install on modern systems.

J2ME DevKit provides a simple, reproducible setup that works with modern development tools such as **VS Code, IntelliJ IDEA, and Eclipse**.

## What the DevKit provides

The project includes:

- a pinned Java ME build toolchain
- automated compilation and preverification
- MicroEmulator for running MIDlets
- a minimal starter MIDlet project

## Repository

The project is available on GitHub:

[J2ME DevKit Repository](https://github.com/retro-systems-lab/j2me-devkit)

Clone the repository and run the setup command to install the toolchain and launch the emulator.
