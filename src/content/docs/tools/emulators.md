# Emulators

Emulators are the fastest way to shorten the feedback loop for old mobile platforms.

Useful things to capture here:

- which SDK bundles its own device emulator
- what screen sizes and input profiles are supported
- where performance differs from real hardware
- how logs, network traffic, and storage can be inspected

For early experiments, prioritize the emulator that matches the target handset family closely enough to validate packaging and runtime behavior.
