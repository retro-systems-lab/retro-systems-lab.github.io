Retro Systems Lab explores the software platforms that powered feature phones in the early 2000s.

These devices ran entire ecosystems of games and applications, many of which have largely disappeared from modern computing culture.

Currently we are exploring **Java ME (J2ME)**, the platform that powered thousands of applications and games on feature phones during the early mobile computing era.

The first project from Retro Systems Lab is the **[J2ME DevKit](/tools/j2me-devkit/)** — a modern development environment for building and running Java ME (MIDlet) applications on contemporary machines.
